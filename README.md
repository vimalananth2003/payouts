# Project Title

💰Payout Dashboard

## Description

A responsive Payout Dashboard UI built using HTML and CSS. The dashboard displays payout information, transaction history, account details, and summary cards with a clean and modern design.

## 🚀 Features

- Responsive layout for desktop and mobile
- Dashboard cards showing payout details
- Payout history table
- Account summary section
- Mobile-friendly design using CSS media queries

## 🛠️ Technologies Used

- HTML
- CSS


## 📁 Project Structure

```
payout-dashboard/
│
├── index.html
├── css/
│   └── style.css
└── README.md
```

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.

## 📱 Responsive Design

The dashboard automatically adjusts its layout for smaller screen sizes using CSS media queries.
